斜率与截距

一条直线如何用 m 与 b 来描述？

课堂情境：初中数学《一次函数》

题目描述

调整斜率 m 和截距 b，观察直线的倾斜程度与在 y 轴上的截距位置。

相关原理

一次函数写作 y = mx + b。m 表示“上升多少/走过多少”，b 表示当 x = 0 时的 y 值。

当斜率为 0 时是水平线；斜率为负时，直线从左上向右下倾斜。

示例分析

  1. m = 1, b = 0：直线穿过原点，倾斜角 45°。
  2. m = -2, b = 3：直线下降较陡，并在 y = 3 处与 y 轴相交。
  3. 斜率的绝对值越大，直线越陡。

拓展练习

  • 将截距设为 0，验证斜率为 1 和 -1 时的对称关系。
  • 找出一条与 x 轴平行的直线，对应的斜率是多少？
  • 当斜率相同、截距不同，两条直线会如何平移？
